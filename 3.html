JavaScript

JavaScript(JS) is a lightweight, interpreter or just-in-time compilation programming language that supports first-class functions.It is well known as a scripting language for web pages, but it is also used in many non-browser environments such as Node.js, Apache CouchDB, and Adobe Acrobat.JavaScript is a prototype-based, multi-paradigm, single thread (en-US), dynamic (en-US) language that supports object-oriented, imperative, declarative (functional programming, etc.) styles.
